# Course Assets

本仓库用于存放长安大学工程机械学院课程学习资料与配套资源，供学习交流使用。

## 资料来源说明

本仓库资料来源分为以下三类：

1. 个人整理
   由仓库维护者本人在学习过程中整理、归纳、撰写的资料。
2. 协作整理
   与同学或朋友共同整理、共同维护的资料，相关贡献将在提交记录或说明中体现。
3. 传承资料
   来自学长学姐或往届同学传承的学习资料。此类资料仅用于学习参考，不代表原作者放弃其权利。

## 使用范围与转载要求

1. 本仓库资料仅供个人学习、复习和教学交流使用。
2. 未经明确授权，不得将本仓库资料二次上传到公众号、知识付费平台、网盘社群、培训引流渠道等。
3. 禁止任何以本仓库资料为核心内容的引流、营销或盈利行为。
4. 如需转载、搬运、整理再发布，请先联系维护者取得书面许可，并保留完整来源说明。
5. 对于来源于他人的资料，使用者应同时遵守原作者或原来源的约束。

## 特别声明（针对公众号搬运）

发现后将保留证据并要求下架。
